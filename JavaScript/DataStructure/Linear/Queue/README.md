# Queue

### Concept

- FIFO(First In First Out)

### 구성 요소

#### Field

- Front
- Rear

#### Method

- EnQueue(): 큐 요소 추가
- DeQueue(): 큐 요소 제거
- peek(): 큐의 가장 앞에 있는 값 확인
- size()

### 종류

- Linear Queue
- Circular Queue
  - Linked List로 구현했을 때 이점이 없다.

### 현실

- 놀이기구 대기열
